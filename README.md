# 微云易购（微信小程序 / miniProgram） #

## 点击[查看项目 API 文档](https://www.showdoc.com.cn/128719739414963?page_id=2513235043485226) ##
### 微信搜索：微云易购，可以查看预览项目。（目前项目还在开发阶段，还未发布到微信小程序官网上 - 2020/11/22） ###

### 安装运行（安装运行前请确定已安装node环境）
* 环境安装：npm install
* 启动服务：npm run dev
* 发布代码：npm run build

### 目前以实现的功能
* 首页
* 搜索框
* 分类
* 分类菜单
* 分类切换
* 分类商品
* 商品列表
* 商品详情
* 收藏商品
* 商品大图显示
* 加入购物车
* 购物车单个数量增加/减少
* 购物车全选/反选
