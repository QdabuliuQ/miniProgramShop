### 安装运行（安装运行前请确定已安装node环境）
* 环境安装：npm install
* 启动服务：npm run dev
* 发布代码：npm run build

### 目前以实现的功能
* 首页
* 搜索框
* 分类
* 分类菜单
* 分类切换
* 分类商品
* 商品列表
